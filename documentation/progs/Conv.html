Program Conv

Usage

Conv 'file' -g -G -m -M
file: matrix_TYP.

Description

Converts the matrix_TYP into the format of other programs available. The options specify the program, and only one of them is allowed at one time.

Options

-g or -G: GAP-format
-m or -M: MAPLE-format
