Program Graph

Usage

Graph 'file1' ['file2'] -[options]
file1: bravais_TYP with REDUCED pointgroup G with CORRECT order
file2: matrix_TYP with presentation for G

Description

Calculates the "graph of inclusions" for a geometric class given by G.
For further information on the output see example 13.

Options

-i    : Print the Z-classes to 'file1.i' and the affine classes
        to 'file1.i.j',
-f    : Calculates the formspace even if it is given.
        CAUTION:
        If you give the formspace and the normalizers,
        they have to be correct. Then G->gen and G->normal
        have to generate the normalizer of G.
-l    : If the order of a cohomology group is >= 2097152,
        you have to use this option; the program needs more
        time with this option.
-d    : only for debugging

Remarks

See also for Extensions/Vector_systems, K_Subgroups, K_Supergroups, Order, Presentation, QtoZ.
