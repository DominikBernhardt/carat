Program K_Subgroups

Usage

K_Subgroups 'file1' 'file2' [-p 'file3'] 'p_1' ... 'p_n'
file1: Spacegroup R in standard affine form
file2: Pointgroup G of R (G->gen and G->normal have to generate the normalizer)
file3: (Optional) Presentation of G
pi : primes

Description

Calculates the maximal klassengleich subgroups of R with pi-power index.

Options

-f   : print the subgroups in files 'file1_j'
-h   : gives some help
-d   : only for debugging, do not use

Remarks

See also for Graph, K_Supergroups and Standard_affine_form.
