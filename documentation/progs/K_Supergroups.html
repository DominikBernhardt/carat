Program K_Supergroups

Usage

K_Supergroups 'file1' 'p_1' ... 'p_n'
file1: Spacegroup in standard affine form
pi : primes

Description

Calculates the minimal klassengleich supergroups with pi-power index.

Options

-t   : omit the last dim generators of the spacegroup
-f   : print the supergroups in files 'file1_j'
-h   : gives some help
-d   : only for debugging

Remarks

See also for Graph, K_Subgroups and Standard_affine_form.
